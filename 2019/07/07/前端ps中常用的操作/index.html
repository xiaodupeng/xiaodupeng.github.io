前端ps中常用的操作

发表于 2019-07-07

首先，打开ps就先来设置一下ps的单位啦
点击最上面的一行的编辑—>首选项–>单位与标尺–>然后设置为像素为单位

把自己需要的图片切下来

第一种方法:
选择切片工具，把自己想要的图片框选出来。然后复制(ctrl+c)，新建(ctrl+n),粘贴(ctrl+v),存储为web所用格式(alt+shift+ctrl+s)
第二种方法：
找到你想要切得图片的对应的图层，点击右键，把其转换为智能对象，然后用矩形框工具把其框起来，然后复制(ctrl+c)，新建(ctrl+n),粘贴(ctrl+v),存储为web所用格式(alt+shift+ctrl+s)

快速找到图层

1.首先点击左边菜单栏移动工具，然后在菜单栏左上角有个“自动选择”选项，勾上。在后面紧接着有一个选项，选择“组”
2.按住alt键不放，鼠标左键点击自己需要的图片，就会自动进入该图层了

ps中一些常用的快捷键

1.ctrl++是放大图片，对应的,ctrl–就是缩小图片啦(alt+鼠标滑轮，也可以进行图片的放大和缩小)
2.ctrl+r 打开辅助线标尺
3.空格键 + 鼠标 就会出来ps的抓手工具
4.ctrl + 0(数字0，不是字母o) 图片会变为ps窗口的大小，也就是你刚刚把图片用ps打开的时候，最原始的状态
5.ctrl + 1(数字1)，图片切换到百分之百显示
6.ctrl + t 自由变换
7.按住shift会等比例缩放图片
8.文字,点击右侧的“T”，，然后选中文字，点击一下文字，文字的信息在ps的最上面的一一些操作的下面就会显示出来了。另外，ctrl + t ，这时候，关于文字的面板信息就会出来了。
9.ctrl+z 是后退一步。ctrl + alt + z可以返回20步以内
10.ctrl + d 取消选区
